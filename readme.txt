Long Description

== Installation ==
- Upload your zip file to your wp-content/plugins directory
- Uncompress the zip file
- Go to the Wordpress dashboard
- Search for _Convertloop for Wordpress_ plugin
- Click on \"Activate\"

== Frequently Asked Questions ==
= Can I use PHP 5.6 =
No. This plugin uses autoloading and declares options that are only supported in PHP 7 or superior

= Do I need to have WooCommerce installed =
No, WooCommerce is not required

= Do I need to have Contact Form 7 =
No, Contact Form 7 is not required

= How can I contribute =
This plugin is Open Source.
You can contribute by creating a pull request in https://github.com/Dazzet/convertloop-wordpress
